# Netflix_Movies_TV_Shows Analysis using Python

This project performs Exploratory Data Analysis (EDA) on the Netflix Titles Dataset using Python.

The objective of this project is to analyze Netflix movies and TV shows, discover patterns, clean the dataset, visualize trends, and gain insights from the data.

---

# Dataset

Dataset Used:
- netflix_titles.csv

Dataset contains information about:
- Movies
- TV Shows
- Directors
- Cast
- Countries
- Ratings
- Genres
- Release Years
- Duration
- Descriptions

---

- Python
